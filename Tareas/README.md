En esta carpeta se encuentran las tareas del curso de Soluciones Numéricas de EDO, que incluyen la implementación y experimentación de los diferentes métodos vistos en clase. 
Cada práctica contiene el código, notas y ejemplos que permiten analizar la precisión, estabilidad y eficiencia de los métodos numéricos, tanto en problemas de valor inicial 
como de frontera.
